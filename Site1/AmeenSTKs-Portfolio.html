Parametric line Sand box #1

Description:

Here is my first project, which as simple processing sketch which takes lines with parametric values for each lines data, My inspiration derives from exploring how parametric data can be implemented in to shapes and how trig functions can create compulsive animations.

I have used the basic programming constructs such as for loops to allow for multiple lines, if statements for user control, trig functions to alter line movements and simple variables to reflect a sandbox like element to the code
